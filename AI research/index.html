Introduction

Hi guys. I set up this page to give some guidance around your use/investigations of ChatGPT.

If you're keen to give this a good play, make sure you use it yourself first so the following questions don't influence your actions/thoughts.

First of all, click on this is link to go to ChatGPT. Create an account if you haven't done so already.

Ideas to use in ChatGPT

  • Ask it to translate something (phrase, sentence, word) from one language to another
  • Ask it to write a story
    • Ask it to write a story in a certain style such as author or structure (Dr. Seuss, poem, Eminem, joke, etc.)
  • Ask it for instructions on something (build a table, cook a roast, etc.)
  • Ask how something works (car, medicine, human body, plane, stock market, etc.)
  • Ask it about history (the release of the first xbox, who invented peanut butter, etc.)
  • Ask it about itself
  • Ask it anything you want!

NOTE: Feel free to give your questions context as well (i.e. why was the creation of the Model T Ford so important, how did the invention of the microprocessor change the world)

Don't forget to have fun and let me know what you think. Enjoy!

What is ChatGPT?

Here is a brief explanation of what ChatGPT is, for those interested:

ChatGPT is a natural language processing chatbot developed by OpenAI, released November 2022.

It is built on the GPT (Generative Pre-trained Transformer) family of large language models. What this means is that it is essentially a large neural network trained on large quantities of unlabelled text using self-supervised learning. This means that it fed a lot of data (300 billion words with 175 billion parameters, from the internet and human feedback) and draws conclusions/relationships itself. It also has supervised/enforced learning from human feedback to curb any errors and minimize bias from the data.

The model is built on a supercomputer infrastructure using Nvidia GPU's with more than 285,000 CPU Cores, 10,000 GPUs and 400 Gigabit per second of network connectivity for each GPU Server. In comparison, a typical laptop will have 4-8 CPU cores, 0-1 GPU cores and maximum 1 Gigabit per second network connectivity.

Its estimated value is $29 billion dollars.

ChatGPT is part of an AI category called narrow AI. Narrow AI is an AI structure that is tasked with a very limited (or narrow) scope, such as chatting, image generation, voice assistants (Alexa), etc.
